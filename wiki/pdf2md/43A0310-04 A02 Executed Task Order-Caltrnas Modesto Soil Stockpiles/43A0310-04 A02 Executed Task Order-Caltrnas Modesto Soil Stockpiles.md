# I. Task Order Purpose

Task Order No. 4, Amendment 1, under Interagency Agreement No. 43A0310, amends the existing Task Order for time and funding. The original Task Order and this amendment provides for the reimbursement of costs incurred by the DTSC for oversight, review, consultation, decision document approval, and/or processing applications(s) or request(s) made by Caltrans to mitigate impacts from the Caltrans Modesto Soil Stockpiles.

# II. Scope of Services

This amendment does not change the scope of services provision of the original Task Order.
